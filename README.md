IGUS E-Chain Selection Assistant

A single offline HTML tool for entering cable/hose specs, auto-packing them around a scaled separator, and picking the smallest fitting IGUS e-chain from your own CSV/Excel catalogue.
Open echain_selector.html directly in any browser — no install, server, or internet required (Excel upload needs internet once for the parser library).
Enter hoses/cables → set gaps & fill limits → upload or load chain data → calculate to get top-3 chain matches with a to-scale cross-section, BOM, and PDF/PNG export.
Drag-zoom, scroll-zoom, and pan the cross-section like a CAD viewer; save/load full projects as JSON to pick up work later.
